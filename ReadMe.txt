Для работы приложения требуется включенный SQL Server! А также база данных.
Устанавливаем приложение.
Также нам понадобится название SQL Server к которому подключаемся, его нужно скопировать (при запуске MSSM в окне есть название сервера).
Открываем папку с установленным приложением (где находится MineLab.exe), открываем файл Connection.txt, и если название вашего сервера отличается от названия которое в нём написано, меняем его, если оно такое же, то оставляем.
Дальше нужно подключиться к SQL Server и зайти в MSSM.
Создать новую БД с названием MineLab.
Чтобы импортировать БД которую вы скачали (файл с расширением .sql), добавьте его приложение путём перетаскивания в него, после чего у вас должен создаться скрипт.
Выполняем скрипт.
Пользуемся приложением.
